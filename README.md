# Sysco (sysco)
Sysco Corporation is the global leader in selling, marketing, and distributing food products to restaurants, healthcare facilities, educational institutions, and hospitality businesses. Sysco operates multiple developer portals including API Central and the APIC Developer Portal, providing APIs for food distribution, ordering, and supply chain integration.

## APIs

### Food Distribution API
The Sysco Food Distribution API provides programmatic access to Sysco's food distribution platform for ordering, product catalog browsing, delivery tracking, and supply chain management. Available through the Sysco API Central developer portal.
